# Changelog

## 1.0.1

- Added English as the default interface language with saved English/Spanish preferences.
- Added a first-run language chooser and a Preferences dialog for changing the interface language.
- Localized menus, dialogs, help text, messages, accessibility labels, and default document text.
- Expanded HTML rule handling with stricter parent-child, content-type, required-field, ordering, and attribute validation based on the builder rule set.
- Added validation before importing, saving, copying, and previewing generated HTML.
- Added link safety normalization for `target="_blank"` and improved `pre` text preservation during import.
- Added an internal rule compliance verifier available with `--verify-rules`.

- First release version.
